Hexo+Github 搭建博客

使用Hexo搭建自己的Github博客

当前环境配置

Windows7 64位

预安装软件

Node.js：Node.js官方下载地址
Git：Git官方下载地址

1、安装Hexo

在认为合适的地方创建一个文件夹，然后在文件夹中打开Git Bash Here。(本文地址：F:\Blog)
安装Hexo:

npm install hexo-cli -g

初始化Hexo： 

hexo init
npm install

之后npm将会自动安装你需要的组件，只需要等待npm操作即可。

创建新文章：

hexo new "Hello World"

生成静态文件：

hexo generate

启动服务器：

hexo server

之后我们就可以通过浏览器打开http://localhost:400/ 来访问：

新建一个github仓库，它要按照如下格式进行命名：你的Github用户昵称.github,io。例如lyc080865.github.io。

然后点击Setting选项，向下翻到GitPage选项

因为当前仓库没有内容所以还是无法访问的，我们可以点击下面的 Choose a theme选择一个主题，
然后将文件提交，之后我们就可以看到：

然后我们就可以通过https://lyc080865.github.io/来访问Github仓库。（若命名不是按照格式来的话，例如：仓库名为lyc.github.io,则访问地址为：https://lyc080865.github.io/lyc.github.io/）

3、配置SSH公钥

配置Github的SSH密钥可以让本地git项目与远程的github建立联系，让我们在本地写了代码之后直接通过git操作就可以实现本地代码库与Github代码库同步。
首先查看我们本地是否存在秘钥： 

~/.ssh

不存在则创建一对新的SSH秘钥（Git的相关配置不再叙述） 

ssh-keygen -t rsa -C "your_email@example.com"

直接三个回车，则秘钥按照默认文件进行存储，也可以输入特定的文件名，文件一般保存在c:\user\账户名.ssh

将id_rsa.pub的内容复制后添加到你的Github账号中的SSH列表中

测试是否设置成功：

ssh git@github.com

或者 hexo d -g；
此时可能会让你输入Github的账号与密码，输入后就可以在浏览器中访问https://lyc080865.github.io/;

5、绑定域名

首先在万网购买域名，然后进入域名与网站管理控制台

进入解析，添加解析（主机记录：一个@ 一个www）

记录类型：`A`提供的是一种IP地址解析到你所买的域名的解析方式。例如：`185.199.108.153` 解析到 `liyuncang.club`; 主机记录：`www`代表一个二级域名此时输入`http://www.liyuncang.club`是可以正常访问;`@`代表直接输入`http://liyuncang.club`是可以访问的; 记录值：指你的Github的地址，可以通过`ping`你的`xxx.github.io`地址获取到;

进入之前的Github仓库，Setting界面，下拉到GitHub Pages，设置Custom domin，输入你的域名，点击save保存。

进入本地博客文件夹，进入source目录，新建一个记事本文件，输入域名，如果带有www，那么以后访问的时候必须带有www完整的域名才可以访问，但如果不带有www，以后访问的时候带不带www都可以访问。所以建议不要带有www。保存，命名为CNAME，没有后缀名。

然后博客目录下Git base here,执行部署：

hexo clean
hexo g
hexo d

之后我们就可以通过域名liyuncang.club来访问你搭建的网站
